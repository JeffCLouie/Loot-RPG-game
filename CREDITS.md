# Credits

## Art

The in-game pixel-art sprites (tiles, walls, floors, water, lava, hero,
enemies, bosses, NPCs, chests, and pickups) come from the **DawnLike** tileset.

- **DawnLike — 16×16 Universal Rogue-like tileset** by **DragonDePlatino**,
  built on the palette by **DawnBringer**.
- License: **CC-BY 4.0** (Creative Commons Attribution 4.0 International).
- Source: https://opengameart.org/content/dawnlike-16x16-universal-rogue-like-tileset-v181

Only the subset of tiles used by the game is bundled, packed into a single
base64 atlas embedded directly in `index.html` to keep the game a single,
self-contained file.

The inventory/equipment panels use the project's own hand-drawn vector icons.
